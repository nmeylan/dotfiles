## Usage

```git clone https://github.com/nmeylan/dotfiles```

```ln -s dotfiles .vim```

```ln -s dotfiles/vimrc .vimrc```

```cd dotfiles```

```git submodule init```

```git submodule update```

## Add bundle (with pathogen)
```git submodule add [bundle-repo] bundle/vim-[bundle-name]```

```git commit -m 'Added vim-[bundle-name]'```
